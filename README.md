# 🚀 Quiz: Tu Primer Despliegue Personalizado

¡Bienvenido al reto! El objetivo de este ejercicio es practicar el flujo básico de trabajo con Git y la edición de archivos HTML. Sigue estos pasos para completar el quiz con éxito.

## 📝 Instrucciones paso a paso

1.  **Clona este repositorio** Abre tu terminal y ejecuta el siguiente comando:

    ```bash
    git clone [URL_DEL_REPOSITORIO]
    ```

2.  **Edita el archivo** Abre el archivo `index.html` en tu editor de código favorito (VS Code, Sublime Text, etc.) y busca la etiqueta `<h1>`.

3.  **Personaliza el contenido** Reemplaza el texto `SU_NOMBRE_AQUI` por:

    - Tu nombre completo.
    - O una frase motivacional que te inspire hoy.

4.  **Captura tu resultado** Abre el archivo `index.html` en tu navegador (para esto puedes usar la extensión Live Server de la que hablamos [click derecho + open with live server]), toma un **pantallazo** del resultado y guárdalo en la carpeta del proyecto con el nombre `resultado.png` (o `.jpg`).

5.  **Actualiza este README** Muestra tu logro reemplazando la imagen de ejemplo de abajo con tu propia captura.
